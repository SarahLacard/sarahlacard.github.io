Here‚Äôs a practical path that works for tools, services, and CLI apps.

A) Decide targets (oldest first)

Pick the oldest suites you care about; build against those to maximize compatibility:

Debian: 12 (bookworm) now; optionally 13 (trixie) later.

Ubuntu: 22.04 LTS (jammy), 24.04 LTS (noble).

Server NIC(s) (Broadcom 1G eno1..eno4, Intel 10G enp6s0f0/1) are for the OS. Connect at least one to your LAN.

Tip: If the iDRAC‚Äôs LCD or web UI shows an address like 192.168.0.120 but your LAN is 192.168.1.0/24, set iDRAC to DHCP and renew; it should pick a correct address from your router/DHCP server.

3) Check Your iDRAC Version and Plan the Upgrade

Open iDRAC ‚Üí Overview ‚Üí iDRAC Settings (or similar) and note Firmware Version.

If < 2.30: first upgrade to 2.30.30.30.

From 2.30: the DUP .EXE for newer versions may still fail with RED007. Two reliable paths exist:

Path A ‚Äî Step Up Through Bridge Versions

Use Dell‚Äôs ‚ÄúiDRAC with Lifecycle Controller‚Äù packages (Windows .EXE DUPs are fine to upload directly in the web UI):

2.30.30.30 ‚Üí

2.40.40.40 ‚Üí

Each hop updates the signing trust chain for the next version.

Path B ‚Äî Bypass the DUP Wrapper (Fix for RED007)

sudo apt install -y p7zip-full

7z x iDRAC-with-Lifecycle-Controller_*2.86*.EXE

This feeds iDRAC the raw firmware image it knows how to verify, avoiding the DUP‚Äôs outer signature envelope that triggers RED007.

Where to watch status: iDRAC Job Queue (Scheduled ‚Üí Verifying ‚Üí Updating ‚Üí Completed). The web UI may briefly disconnect while the controller restarts; reconnect once it‚Äôs back.

4) Enable the HTML5 Virtual Console

9) Power Supply Options (R730 quick guidance)

Redundancy Policy: Enabled (you‚Äôve got dual PSUs‚Äîkeep fault tolerance).

10) Common Errors and Quick Fixes

RED007: Unable to verify Update Package signature

Use bridge versions (2.30 ‚Üí 2.40 ‚Üí 2.50/2.52 ‚Üí 2.86), or extract the DUP and upload payload/firmimg.d7.

JCP003: Repository Update failed at 0%

Very old iDRAC can‚Äôt negotiate modern HTTPS with Dell. Manual upload or firmimg.d7 works. Once you‚Äôre on newer firmware, repository updates work again.

Java/handshake errors launching the old console (TLS/ciphers)

Skip all of that. Upgrade iDRAC and switch to HTML5 Virtual Console.

Virtual Console opens but shows a blank canvas

Clear browser cache for the iDRAC host or use Troubleshooting ‚Üí Reset iDRAC to restart the BMC services.

Installer falls into PXE repeatedly

Map the ISO again and use F11 ‚Üí Boot Manager ‚Üí Virtual CD/DVD. Ensure Virtual Media bar says the ISO is connected.

Debian installer can‚Äôt see disks

Create a Virtual Disk in Ctrl‚ÄëR (or switch PERC to HBA).

11) Pre‚ÄëPublish Redaction Checklist (if you share logs)
